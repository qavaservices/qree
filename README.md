# Customer Success Manager

A monochrome web application for managing customer feedback, tracking Linear tickets, and organizing meeting notes.

## Development Stages

This project is being built in progressive stages. Currently: **Stage 1 - UI Skeleton**

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- React Query (for future stages)
